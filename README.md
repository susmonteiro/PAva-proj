## Compile and Run Project 1

```
    $ javac HardWorkers.java

    $ javac -classpath javassist.jar:. ist/meic/pava/UsingMethodCombination.java

    $ java -classpath javassist.jar:. ist.meic.pava.UsingMethodCombination HardWorkers
```
